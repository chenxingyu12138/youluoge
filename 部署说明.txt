游略阁 — 免费部署与访问统计配置说明
================================

【重要】国内访问说明
--------------------------------
Vercel 域名（*.vercel.app）在中国大陆经常无法打开或极慢，
这是网络环境限制，不是网站代码问题。

解决方案（任选其一）：
  A. 改用 Cloudflare Pages 部署（推荐，见下方第二节）
  B. 修改 DNS 为 223.5.5.5 或 114.114.114.114 后重试 Vercel 链接
  C. 使用 VPN 访问 Vercel 域名


GitHub 仓库
--------------------------------
https://github.com/chenxingyu12138/youluoge


一、Cloudflare Pages 部署（国内推荐，免费）
--------------------------------

1. 注册 Cloudflare
   打开 https://dash.cloudflare.com 注册/登录（免费）

2. 导入 GitHub 项目
   - Workers & Pages → Create → Pages → Connect to Git
   - 选择 GitHub 账号，授权后选中 youluoge 仓库
   - 构建设置：
     Framework preset : None
     Build command    : （留空）
     Build output     : /
   - 点击 Save and Deploy，约 1-2 分钟完成

3. 获得访问地址
   形如：https://youluoge.pages.dev
   国内一般可正常访问，可直接分享给任何人

4. 开启访问统计
   a) 创建 KV 命名空间
      Workers & Pages → KV → Create namespace
      名称：youluoge-stats

   b) 绑定到 Pages 项目
      Pages → youluoge → Settings → Functions
      → KV namespace bindings → Add binding
      Variable name : STATS_KV
      KV namespace  : youluoge-stats

   c) 添加环境变量
      Settings → Environment variables → Add
      Variable : ADMIN_PASSWORD
      Value    : 你的管理密码

   d) 重新部署
      Deployments → 最新 → ... → Retry deployment

5. 查看访问数据
   https://youluoge.pages.dev/admin
   输入 ADMIN_PASSWORD 登录


二、Vercel 部署（海外访问友好，免费）
--------------------------------

1. 安装 GitHub 集成
   https://github.com/apps/vercel → Configure
   授权访问 youluoge 仓库

2. 一键导入
   https://vercel.com/new/import?s=https://github.com/chenxingyu12138/youluoge
   Framework: Other，Root: ./，直接 Deploy

3. 访问地址
   https://youluoge.vercel.app
   （国内可能无法访问，见顶部说明）

4. 开启统计（Vercel KV）
   Storage → Create Database → KV → Connect to Project
   Environment Variables → ADMIN_PASSWORD = 你的密码 → Redeploy
   后台：https://youluoge.vercel.app/admin


三、常见问题
--------------------------------

Q: youluoge.vercel.app 打不开？
A: 国内网络对 vercel.app 限制较多。请改用 Cloudflare Pages（第一节），
   或切换 DNS 到 223.5.5.5 后重试。

Q: 部署后统计一直是 0？
A: 确认 KV 已绑定（Cloudflare: STATS_KV / Vercel: KV_REST_API_*），
   且已 Redeploy；浏览几个页面后再刷新 admin。

Q: admin 提示密码错误？
A: 检查 ADMIN_PASSWORD 环境变量是否设置，重新部署后再试。

Q: 想用自定义域名？
A: Cloudflare Pages 或 Vercel 的 Settings → Domains 均可免费绑定。

Q: 两个平台都部署了，统计会重复吗？
A: 会各自独立统计，建议只对外分享一个主域名。
