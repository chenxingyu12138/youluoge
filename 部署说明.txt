游略阁 — 免费部署与访问统计配置说明
================================

【重要】国内访问说明
--------------------------------
Vercel 域名（*.vercel.app）在中国大陆经常无法打开或极慢。
推荐改用 Cloudflare Pages（*.pages.dev），国内一般可正常访问。

GitHub 仓库：https://github.com/chenxingyu12138/youluoge


一、Cloudflare Pages 部署（国内推荐）
--------------------------------

⚠️ 部署失败 Asset too large 的原因：
   创建了 Worker 项目，或 Build command 填了「npx wrangler deploy」，
   导致把整个目录（含 node_modules）当作静态文件上传。

正确步骤：

1. Cloudflare 控制台 → Workers & Pages → Create application
   → 选择 **Pages** 标签（不要选 Worker！）
   → Connect to Git → 选择 youluoge 仓库

2. 构建设置（必须这样填）：
   Framework preset : None
   Build command    : （完全留空！）
   Build output     : /
   Root directory   : /

3. Save and Deploy，等待 1-2 分钟

4. 若已有项目失败，修改设置后重试：
   Pages → youluoge → Settings → Builds
   → 清空 Build command → Save
   → Deployments → Retry deployment

5. 访问地址：https://youluoge.pages.dev

6. 开启访问统计：
   a) KV → Create namespace（名称 youluoge-stats）
   b) Pages → Settings → Functions → KV bindings
      Variable: STATS_KV → 选择 youluoge-stats
   c) Settings → Environment variables
      ADMIN_PASSWORD = 你的管理密码
   d) Deployments → Retry deployment

7. 统计后台：https://youluoge.pages.dev/admin


二、Vercel 部署（海外访问友好）
--------------------------------

1. 安装 GitHub 集成：https://github.com/apps/vercel
2. 导入：https://vercel.com/new/import?s=https://github.com/chenxingyu12138/youluoge
3. 访问：https://youluoge.vercel.app（国内可能打不开）
4. 统计：Storage → KV → Connect → 设置 ADMIN_PASSWORD → Redeploy
5. 后台：https://youluoge.vercel.app/admin


三、常见问题
--------------------------------

Q: Cloudflare 报 Asset too large / workerd 119 MiB？
A: Build command 必须留空。不要用 Worker 项目，要用 Pages 项目。

Q: youluoge.vercel.app 打不开？
A: 国内网络限制，请用 Cloudflare Pages 域名分享。

Q: 统计一直是 0？
A: 确认 KV 已绑定并 Redeploy，浏览几个页面后再刷新 admin。

Q: admin 密码错误？
A: 检查 ADMIN_PASSWORD 环境变量，重新部署后再试。
